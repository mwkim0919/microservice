# microservice
Personal project for microservice

### Config-Server
```
spring.application.name=<configuration>
server.port=8888
```
*spring.application.name* value has to match with *configuration.yml* to fetch configuration values

### Discovery-Server
```
server.port=8761
```

### API-Gateway
```
server.port=8080
localhost:8080/swagger-ui.html
```
